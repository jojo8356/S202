        la prochaine étape.
        La classe de tests SearchEngineTests.




  La classe IndexedPage
  La classe IndexedPage représente une page web sous forme indexée.

  Attributs
        String url : l'url de la page qui a été indexée
        String[ ] words : un tableau (array) de chaînes de caractères. Les tableaux en Java ressemblent
        beaucoup aux tableaux C (on verra plus tard qu'il existe d'autres conteneurs plus élaborés en Java,
        mais pour l'instant nous utiliserons des tableaux "de base"). Par rapport au C, ils présentent tout de
        même quelques améliorations notables :
             ils sont initialisés (à 0 pour des entiers, 0.0 pour des flottants, null pour des objets...).
             on a accès à leur taille via leur attribut .length
             on ne peut pas sortir par erreur du tableau (une exception est levée en ce cas)



  Constructeurs
        IndexedPage(String[ ] lines) : construction à partir d'un tableau de chaîne de caractères.
        La première chaîne du tableau correspondra à l'URL de la page, les autres chaînes seront de la forme
        "mot:nombre d'occurences".
        Exemple d'utilisation :
        IndexedPage(new String[] {"http://example.org", "hello:1", "world:1", "java:7"})
        Cela signifie que le document indexé a pour url "http://example.org" et contient 1 fois le mot hello, 1
        fois le mot world et 7 fois le mot java.
        IndexedPage(String text) : construction à partir d'un texte. Ce sera utilisé pour convertir la requête en
        objet IndexedPage lorsqu'on lancera une recherche.
        Il faut donc parcourir le texte et compter le nombre de fois que chaque mot apparaît. Vous pouvez
        utiliser la méthode String.split(String regex)  pour découper la chaîne et Arrays.sort(Object[ ]
        a)  pour trier les mots.



  Méthodes
        String getUrl() : renvoie l'url.
        double getNorm() : calcule et renvoie la norme du vecteur à partir du nombre d'occurrences de chaque
        mot (relire au besoin le TP python sur le modèle vectoriel).
        int getCount(String word) : renvoie le nombre d'occurrences du mot word dans le document.

        double getPonderation(String word) : renvoie la valeur de la coordonnée correspondant au mot
        word dans le vecteur normalisé correspondant au document indexé (à ce stade, on implémente
        seulement le modèle vectoriel basique, sans utiliser de logarithme).
        double proximity(IndexedPage page) : on calcule la similitude entre deux pages, c'est-à-dire le
        produit scalaire du vecteur de la page et du vecteur de l'autre page. Remarque : en fait, on n'a pas
        besoin de générer vraiment les vecteurs pour calculer le produit scalaire, puisque les coordonnées
        nulles ne nous intéressent pas. Il suffit d'effectuer la somme du produit des coordonnées uniquement
        pour les mots apparaissant dans les deux documents.
        String toString(): renvoie "IndexedPage [url=http://example.org]" dans le cas de l'exemple précédent.
